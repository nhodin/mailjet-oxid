mailjet-oxid
============

Plugin Mailjet pour Oxid eShop (v4.7+/5.0+)

Synchronisez votre liste de contacts Mailjet avec vos abonnés newsletter et configurez l'envoi des emails.

Install
=======

Pour installer le module, copiez twmailjet dans le dossier "modules" de l'installation de votre boutique Oxid. 
Puis, allez dans le menu "Extensions" > "Modules" pour activer le module "Connecteur Mailjet - Take it Web".

Un nouveau menu "Mailjet" apparaît après rafraîchissement du BO.
